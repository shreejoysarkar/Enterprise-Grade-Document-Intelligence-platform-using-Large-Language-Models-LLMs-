# Enterprise-Grade-Document-Intelligence-platform-using-Large-Language-Models-LLMs-
 An AI system that ingests unstructured enterprise documents and converts  them into structured, query-able knowledge.

## 🚀 How to Run  

Follow these steps to set up and run the project locally:

---

### 1️. Clone the Repository  

git clone : https://github.com/shreejoysarkar/Enterprise-Grade-Document-Intelligence-platform-using-Large-Language-Models-LLMs-.git


### 2. Create a uv enrionment after opening the repository


uv init --name doc_intel


uv venv doc_intel --python 3.13


Activating the env

doc_intel\Scripts\activate
